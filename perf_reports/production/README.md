# Production Performance Reports

This directory will contain raw logs, CSV/JSON metrics, and charts from the multi-device verification campaign.

Structure
- perf_reports/production/YYYYMMDD-HHMM/
  - logs/* (raw device logs, monitor logs)
  - metrics/*.csv (latency, CPU, memory, battery)
  - charts/*.png (aggregated graphs)
  - notes.md (operator notes)
